Installation

Rhino Plugin — Mesh2Srf

  1. Скопіюй всі файли з секції "Rhino Plugin" нижче
  2. На Windows створи папку:
    %APPDATA%\McNeel\Rhinoceros\8.0\Plug-ins\PythonPlugIns\Mesh2Srf\dev\
  3. Збережи кожен файл у цю папку з правильним ім'ям
  4. Перезапусти Rhino 8
  5. Набери M2SPanel в командному рядку

Grasshopper Scripts

  1. Відкрий Grasshopper
  2. Постав Script компонент (Python 3)
  3. Двічі клікни на ньому
  4. Скопіюй код зі сторінки (кнопка Copy) і встав
  5. Підключи Number Sliders до входів
